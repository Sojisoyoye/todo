# todo
A Todo API in Go.

To run the app:

```
go run cmd/server/main.go
```

**API Endpoints**

- POST /api/v1/todos - Create a new todo

- GET /api/v1/todos - Get all todos

- GET /api/v1/todos/:id - Get a specific todo

- PUT /api/v1/todos/:id - Update a todo

- DELETE /api/v1/todos/:id - Delete a todo
